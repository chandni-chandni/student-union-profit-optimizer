Student Union T-Shirt Profit Optimizer

Java Swing Desktop Application for Profit Optimization & Break-Even Analysis

Project Overview

Developed a Java-based desktop application using Java Swing to optimise student union t-shirt production and maximise profit using mathematical modelling.

The application performs:

  • Optimal production quantity calculation
  • Break-even analysis
  • Sensitivity analysis
  • Revenue and cost forecasting
  • JUnit testing and validation

Technologies Used

  • Java
  • Java Swing GUI
  • Object-Oriented Programming (OOP)
  • JUnit Testing
  • Mathematical Optimization
